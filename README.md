# Spark Agent Showcase

A sanitized public showcase of an operator-facing options trading system built around iron condor workflows, deterministic rules, live/paper account separation, and a monitoring + approval dashboard.

## Stack
- Python
- FastAPI
- Next.js
- PostgreSQL / Timescale-style schema
- Rules-based strategy engine
- LLM-assisted trade commentary layer

## What this repo shows
- Candidate generation and scoring
- Position and lifecycle tracking
- Approval and monitoring UI
- Paper/live execution separation
- Reconciliation, health, and risk workflows

## Notes
This repository is a sanitized showcase build. Secrets, tokens, personal notes, and private runtime state have been removed or redacted.
